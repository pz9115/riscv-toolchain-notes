# Using FFmpeg as a Benchmark to Compare GCC and LLVM

- Event: GNU Tools Cauldron 2025
- Topic: RISC-V GCC/LLVM performance comparison using FFmpeg workloads
- Language: English
- Format: Public talk slides
- Video: https://www.youtube.com/watch?v=BGns00WdfMQ&t=44s
- Slides: [slides.pptx](./slides.pptx)

## Abstract

This talk uses FFmpeg workloads as a real-world benchmark to compare GCC and LLVM on RISC-V hardware. It covers the cross-compilation setup, benchmark input preparation, board testing methodology, performance results, and follow-up questions around RVV, LMUL, code size, and debug information size.

## Notes

The original slides were presented at GNU Tools Cauldron 2025 and have also been publicly shared through the event materials.

The benchmark results reflect the toolchain versions, board, workload, and test setup used at the time of the talk. They should not be interpreted as a general or current performance claim for all GCC/LLVM versions or all RISC-V platforms.

This directory records public talk materials and related notes for long-term technical reference.
